# React Lazy Hydration

A React component for lazy hydration of components.

## Installation

```bash
pnpm add react-lazy-hydration
```

## Usage

```tsx
import { LazyHydrate } from 'react-lazy-hydration';

function App() {
  return (
    <LazyHydrate>
      <YourComponent />
    </LazyHydrate>
  );
}
```
